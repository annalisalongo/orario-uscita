# Calcolo Orario di Uscita

Mini app Streamlit per calcolare l'orario di uscita.

## Regole
- Lavoro fisso: **7 ore e 38 minuti**
- Pausa pranzo selezionabile: **30 minuti** oppure **1 ora**
- L'utente inserisce solo l'**orario di inizio**
- Uscita minima: **16:38**

## Avvio locale
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Deploy su Streamlit Community Cloud
1. Carica questi file su GitHub
2. Vai su Streamlit Community Cloud
3. Crea una nuova app
4. Seleziona il repository
5. Scegli `app.py` come file principale
6. Fai deploy
